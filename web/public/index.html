About Melody

Melody is a Linux systems infrastructure monitoring tool written in Go. The data you see above are real-time system metrics gathered from the server running cadenceradio.com.

Data is measured on the host server by an agent, which reports new measurements to a central API server. This webpage polls the API for the latest system data.

The project is still in an early state. For feedback, feel free to contact me.
