Melody is a real-time Linux monitoring tool written from scratch. It is a proof of concept by Ken Ellorando.

The data above are metrics gathered by an agent monitoring the server running cadenceradio.com. These metrics are relayed to an API server which is continuously polled by your browser.

If you do not see any metrics updating, you may need to enable JavaScript in your browser.
